# TYPO3 CMS 

Erstelle eine neue SQLite3-Datenbank mit PHP:

```bash
sudo apt-get install php-sqlite3
sudo systemctl restart nginx
```

Oder erstelle eine neue PostgreSQL-Datenbank:

```bash
sudo -u postgres -i
createdb -E UTF8 -O thorsten typo3
exit
```
